# Препроцессор

## Описание
**Препроцессор** - это инструмент на базе C++, использующий регулярные выражения для нахождения файлов и импорта данных из них в итоговый проект.

## Возможности
* Весь функционал **препроцессора** сосредаточен в функции *Preprocess(...)*, которая на основе пути к исходному файлу, считывает из него данные и выгружает их в итоговый файл.
* Если в каталоге с исходным файлом не найдены сторонние библиотеки, которые указаны в самом исходнике, то их поиск осуществляется по контейнеру с включенными каталогами.

## Использование
Для использования **препроцессора** необходимо вынести функции *Preprocess(...)* и *FindDirectory(...)* в отдельный файл и импортировать его как стороннюю библиотеку.
